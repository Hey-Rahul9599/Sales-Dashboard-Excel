# Sales-Dashboard-Excel

## Project Overview
This Sales Dashboard is built using Microsoft Excel to analyze sales performance and business trends. The dashboard provides interactive visualizations and key performance indicators (KPIs) to help track revenue, profit, orders, and customer insights.

## Features
- Total Sales Analysis
- Profit & Revenue Tracking
- Monthly Sales Trends
- Region-wise Performance
- Product Category Analysis
- Interactive Charts and KPIs
- Data Cleaning and Transformation

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

## Dataset
The dataset contains sales transactions including:
- Order Date
- Product Category
- Region
- Sales
- Profit
- Quantity

## Key Insights
- Identified top-performing products and regions.
- Analyzed monthly sales growth trends.
- Tracked overall business performance using KPIs.
